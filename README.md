# 1. Project Overview (프로젝트 개요)
- 프로젝트 이름: Text-To-SQL
- 프로젝트 설명: DB지식이 없는 비전공자들을 위한 자연어-SQL 번역 시스템

# 3. Key Features (주요 기능)
- **자연어 질의 입력**:
  - 예시 : "학생 테이블에 있는 학번 데이터 조회해줘"

- **SQL 구문 생성**:
  - 예시 : "SELECT STUDENT_ID FROM STUDENT"
